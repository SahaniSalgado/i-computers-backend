💻 i-computers-backend

Backend API for the iComputers application built using the MERN stack. This project handles user authentication, product management, and order processing for an online computer store.

🚀 Tech Stack
MongoDB – Database
Express.js – Backend framework
Node.js – Runtime environment
Mongoose – ODM for MongoDB
JWT (JSON Web Token) – Authentication
bcrypt.js – Password hashing


📂 Project Structure
i-computers-backend/
│── config/          # Database connection & configs
│── controllers/     # Business logic
│── models/          # Mongoose schemas
│── routes/          # API routes
│── middleware/      # Custom middleware (auth, error handling)
│── utils/           # Helper functions
│── server.js        # Entry point
│── .env             # Environment variables
